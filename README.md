# FuniBuni Android App

<앱 사진>

AI 기반 대형폐기물 처리 원터치 서비스 **FuniBuni** 안드로이드 앱 입니다.


## Service Flow

<Flow 사진>

1. 지역 설정 (처음 이용시에만)
2. 폐기물 사진 촬영 및 업로드
3. 폐기물 카테고리 자동 분류 (AI-based)
4. 사이즈 입력
   - AR 기반 측정
   - 수기 입력
5. 바로 배출 or 나눔 후 배출 선택
